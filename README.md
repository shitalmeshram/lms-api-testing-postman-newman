# LMS API Testing Project

## 📌 Project Overview
This project demonstrates end-to-end API testing for the Learning Management System (LMS) Batch Module.

## 🔧 Tools & Technologies
- Postman
- Newman
- JavaScript (Postman scripts)
- REST APIs

## 📂 Modules Tested
- Create Batch (POST /batches)
- Get All Batches (GET /batches)
- Get Batch by ID
- Get Batch by Name
- Update Batch
- Delete Batch

## ✅ Test Scenarios
- Positive and Negative test cases
- Status code validation (200, 201, 400, 404)
- Schema validation
- Data validation
- Authentication handling

## 📊 Automation
- Automated using Newman CLI
- HTML reports generated using newman-reporter-htmlextra

## 🚀 Key Achievements
- Validated API responses and ensured data accuracy
- Improved API reliability through structured test cases
- Automated regression testing using Newman
